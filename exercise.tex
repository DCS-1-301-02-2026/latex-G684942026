\section{考察}
% ーーー
% 見出し(3)
% 考察
%
% \begin{itemize}
% \end{itemize}
% を使って箇条書きで記述する

図\ref{fig:競争力},表\ref{fig:加入者数}より,次のことが考えられる.
\begin{itemize}
    \item デジタル競争力ランキングと光ファイバー加入者数の直接的な関係は薄い
    \item 国民の娯楽の中心がインターネットを使用する国は、光ファイバーの加入者数が多い
    \item アメリカには他国から技術者が集まるので、総合順位が高い
    \item 光ファイバー回線の加入者数は、現在の国民のデジタルへの関心と相関がある
\end{itemize}
